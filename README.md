## ğŸ¯ Customization

### 1. Update Personal Information
Replace the following placeholders in `index.html`:
- `Your Name` â†’ Your actual name
- `YOUR-LINKEDIN` â†’ Your LinkedIn username
- `YOUR-GITHUB` â†’ Your GitHub username
- `your.email@example.com` â†’ Your email address
- Update the "About Me" section with your story

### 2. Add Your Profile Photo
Replace the placeholder image URL with your photo:
```html
<!-- In the About section -->
<img src="images/profile.jpg" alt="Profile Photo" class="profile-img">
```

### 3. Customize Colors
Edit the CSS custom properties in `css/style.css`:
```css
:root {
    --primary: #0066FF;        /* Change to your preferred color */
    --secondary: #00D9FF;
    --accent: #FF6B6B;
    /* ... other colors */
}
```

### 4. Add Your Projects
Update the projects section in `index.html` with your actual projects. Replace "Coming Soon" cards with real projects as you build them.

### 5. Add Your Resume
Place your resume PDF in the `assets/` folder and ensure it's named `resume.pdf`.
